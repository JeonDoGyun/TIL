## variables ##
var : 변수 생성 키워드
- ex) 길이가 3인 정사각형의 넓이
var area = length * length
print(area)

- var 키워드는 선언할 때만 사용. 값 변경할 때는 사용 안함.
length = 5
print(length)

- var : 데이터 저장소. 식별 이름으로 값을 저장하고 참조. 변수에 값을 할당하기 위해선 할당 연산자가 필요.

- exercise : variable assignment
var x = 5
var y = 9
var z = x + y
print(z)

var length = 12
var width = 7
var area = length * width
print(area)
