# Xcode Playground #
- 개발 환경 종류
- 실시간으로 코드의 결과를 확인할 수 있음.

1. 사용방법
Xcode - File - new - playground - next - 이름 설정, create

- 플레이버튼을 누르면 실시간으로 확인 가능
- 실행방법 변경 : 왼쪽 하단에 있는 플레이버튼 길게 누르면 자동 실행으로 변경 가능
